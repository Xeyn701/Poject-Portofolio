Tentang Saya

Programer & Network Engineer

Halo nama saya Rio Imam Pradana tinggal di provinsi Kalimantan Selatan. Saya sangat suka dengan hal yang berkaitan dengan Teknologi termasuk Pemprograman. Saya Memiliki beberapa Keahlian seperti Instalasi Jaringan, Perakitan Komputer, Serta beberapa Bahasa pemprograman. Awalnya, saya sangat tidak suka hal yang berkaitan dengan pemrograman. Ketika saya memulai mencari dan melaksanakan project yang merupakan sebuah pembelajaran dari kampus, ada client yang meminta membuatkan sebuah website, disitu saya mulai bimbang dengan hal itu karena saya adalah seseorang yang amat tidak tahu tentang dunia pemprograman, tapi apabila permintaan itu ditolak, saya dan kelompok saya akan kesulitan lagi dalam mencari project sedangkan deadline untuk mendapatkan project sudah dekat sehingga dengan terpaksa project pembuatan website kami terima. Hari demi hari menuntun saya untuk memaksakan diri mempelajari dunia pemrograman dari modul-modul dari dosen yang sudah diajarkan, saya juga mulai mencari webinar tentang pemprograman. Beberapa minggu berlalu, saya mulai menyukai dengan pemprograman dan membuat rasa ingin tahu saya semakin kuat dengan ini. Saya mulai banyak mencari webinar-webinar yang berhubungan dengan pemprograman, mulai dari pemprograman web, aplikasi mobile, hingga pemprograman game menggunakan C#. Saat ini saya mengikuti beberapa pelatihan mulai dari pembuatan website dan aplikasi dan juga saya mengikuti Studi Independen MSIB dari kemendikbud untuk membuat Game.
